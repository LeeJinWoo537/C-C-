# 💻©️ **C언어 다른 자료형들~**

| **자료형**   | **크기**  | **범위**               | **동일 자료형**        | **설명**                           |
|--------------|-----------|------------------------|------------------------|-----------------------------------|
| `uint8_t`    | 8비트 (1바이트)  | 0 ~ 255               | `unsigned char`        | 부호 없는 8비트 정수                 |
| `uint16_t`   | 16비트 (2바이트) | 0 ~ 65,535            | `unsigned short`       | 부호 없는 16비트 정수                |
| `uint32_t`   | 32비트 (4바이트) | 0 ~ 4,294,967,295     | `unsigned int`         | 부호 없는 32비트 정수                |
| `uint64_t`   | 64비트 (8바이트) | 0 ~ 18,446,744,073,709,551,615 | `unsigned long long`  | 부호 없는 64비트 정수                |
